JAVA SPRING MICROSERVICES with SYNCHRONOUS Communication

This is the basic project from which the other projects have been derived from. The project consists of 3 microservices
(product-composite, product-recommendation and product-service) and general directory (api) for interfaces and models.

No data persistences have been added and data is stored in memory.

Summary:
- Java 17
- Spring Boot
- Docker
- Gradle

Testing:
- start the project with ". docker_up.sh"
- type in "http://localhost:8080/composite/12" at your browser and the response will be a JSON aggregated product
